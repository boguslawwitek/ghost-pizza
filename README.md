# Overview
A simple website for GhostPizza pizzeria.
Demo: [ghost-pizza.pl](https://ghost-pizza.pl).

# Technologies
- Next.js

# Installation
```
git clone https://github.com/boguslawwitek/ghost-pizza.git
cd ghost-pizza
npm install
npm run dev
```
#### Production
```
npm run build
npm start
```
